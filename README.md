# 🌟 Biography Web Page

A simple and attractive biography webpage built using HTML and CSS.

## 📖 Project Overview

This project is a biography webpage that presents information about a famous personality. It was created to practice HTML structure and CSS styling.

## ✨ Features

- Clean and attractive design
- Easy-to-read layout
- Biography section
- Achievements section
- Responsive styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- Git
- GitHub

## 📂 Project Structure

```text
Biography_html/
│
├── index.html
└── README.md
```

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in your web browser.
